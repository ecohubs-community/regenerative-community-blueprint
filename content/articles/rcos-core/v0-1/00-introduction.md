## 0.1 Purpose of RCOS

Defines a standardized, modular operating system for small, intentional communities focused on resilience, fairness, and regeneration.

## 0.2 Scope of the Core

This specification defines the **mandatory core layers** required for RCOS compliance. Domain-specific practices are excluded and addressed via optional modules.

## 0.3 Design Principles

- Constraint-based, not value-based
- Pre-commitment over improvisation
- Modular by default
- Human-scale
- Failure-tolerant
